# Images du diaporama

## Logo Crafting Tech (fond transparent)

Pour que le blanc du logo devienne transparent sur la diapo :

1. **Télécharge** le logo : https://craftingtech.fr/assets/logo-ClPFe5ny.png  
2. **Enlève le fond blanc** sur [remove.bg](https://remove.bg) (glisse l’image, télécharge le PNG).  
3. **Enregistre** le fichier sous le nom : **craftingtech-transparent.png**  
4. **Dépose-le** dans ce dossier `public/`.

Le logo s’affichera alors avec un fond transparent sur la diapositive.
